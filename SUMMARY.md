# Proposal Diagram Copilot - MVP 项目总结

## 项目概览

已完成 MVP 版本的方案图转换工具，实现从图片/PDF 到可编辑 PPTX 的端到端转换流程。

## 核心文件

### 1. API 服务 (`services/api/src/main.py`)

**功能：**
- FastAPI 应用，提供 RESTful API
- 内置内存存储（Redis 可选）
- 异步 Worker 模拟推理流程
- 生成示例 PPTX 文件

**API 端点：**
- `GET /v1/health` - 健康检查
- `POST /v1/assets/upload-session` - 创建上传会话
- `POST /v1/assets/upload/{asset_id}` - 上传文件
- `POST /v1/jobs` - 创建转换任务
- `GET /v1/jobs/{job_id}` - 查询任务状态
- `GET /v1/jobs/{job_id}/result` - 获取任务结果
- `GET /v1/jobs/{job_id}/download` - 下载 PPTX

**Worker 流程：**
```
queued → preprocessing → inferencing → reconstructing → scoring → done
  0%        15%              45%              75%             95%    100%
```

### 2. Web 前端 (`apps/web/src/app/page.tsx`)

**功能：**
- 文件上传（支持 PNG/JPG/PDF）
- 实时进度显示（轮询 1s）
- 质量评分展示
- PPTX 下载

**技术栈：**
- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- 原生 fetch API

### 3. 共享类型 (`packages/shared-types/src/shared_types/__init__.py`)

**定义的模型：**
- `JobStatus` - 任务状态枚举
- `OutputFormat` - 输出格式枚举
- `Job` - 任务模型
- `Asset` - 资产模型
- `UploadSessionRequest/Response`
- `CreateJobRequest/Response`
- `JobStatusResponse`
- `JobResult`
- `QualityScore`

## 运行方式

### 方式 1: 一键启动（推荐）

```bash
./start.sh
```

首次运行会自动：
1. 创建 Python 虚拟环境
2. 安装 Python 依赖（FastAPI, Pydantic, python-pptx, Pillow）
3. 安装 Node.js 依赖（Next.js, React, Tailwind）
4. 启动 API 服务（端口 8000）
5. 启动 Web 服务（端口 3000）

访问：http://localhost:3000

### 方式 2: 手动启动

```bash
# 终端 1 - API 服务
cd services/api
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
python -m uvicorn src.main:app --reload --port 8000

# 终端 2 - Web 服务
cd apps/web
npm install
npm run dev
```

### 方式 3: 快速启动（已安装依赖）

```bash
./dev.sh
```

## 下一步建议

### 短期（1-2 周）

1. **集成真实推理引擎**
   ```python
   # workers/gpu-engine/src/worker.py
   def process_job(job_id: str):
       # 1. 下载原始图片
       image = download_asset(job.asset_id)
       
       # 2. SAM3 分割
       segments = sam3_model.segment(image)
       
       # 3. OCR 识别
       texts = ocr_engine.recognize(image, segments)
       
       # 4. 结构重建
       pptx = pptx_builder.build(segments, texts)
       
       # 5. 质量评分
       score = quality_scorer.evaluate(pptx, image)
       
       return pptx, score
   ```

2. **Redis 队列**
   ```python
   # 替换内存队列
   import redis
   r = redis.Redis()
   r.lpush("jobs:queue", job_id)
   job_id = r.brpop("jobs:queue", timeout=5)
   ```

3. **Postgres 数据库**
   ```sql
   CREATE TABLE jobs (
       id UUID PRIMARY KEY,
       project_id UUID REFERENCES projects(id),
       status VARCHAR(20),
       progress INT,
       created_at TIMESTAMP,
       updated_at TIMESTAMP
   );
   ```
