# Proposal Diagram Copilot v0.1.0-mvp

首个可运行 MVP 版本，已打通端到端闭环：

**上传文件 -> 创建任务 -> 阶段进度 -> 质量评分 -> 下载 PPTX**

## ✨ Highlights

- 可上传图片/PDF 并创建转换任务
- 任务状态机可视化：`queued -> preprocessing -> inferencing -> reconstructing -> scoring -> done`
- 输出质量评分：
  - editableRate
  - ocrAccuracy
  - layoutDeviation
  - score
- 可下载示例 PPTX 结果文件
- API 文档可直接访问（Swagger）

## 🧱 Architecture (MVP)

- **Web**: Next.js 15 + React 19 + Tailwind
- **API**: FastAPI
- **Queue**: 内存队列（Redis 预留）
- **Worker**: 模拟 Pipeline Worker（真实推理待接入）

## 📁 Repository Structure

- `apps/web` 前端上传/进度/下载页面
- `services/api` FastAPI 接口与任务编排
- `workers/gpu-engine` GPU 推理服务预留目录
- `packages/shared-types` 共享类型定义

## 🔌 API Endpoints

- `GET /v1/health`
- `POST /v1/assets/upload-session`
- `POST /v1/assets/upload/{asset_id}`
- `POST /v1/jobs`
- `GET /v1/jobs/{job_id}`
- `GET /v1/jobs/{job_id}/result`
- `GET /v1/jobs/{job_id}/download`

## 🚀 Quick Start

```bash
./start.sh
# or
./dev.sh
```

- API: `http://localhost:8000`
- API Docs: `http://localhost:8000/docs`
- Web: `http://localhost:3000` (若冲突自动切换 3001)
